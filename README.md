# Bravo Generators

My personal Rails generator scripts inspired by [nifty-generators]

## Dependency

* Ruby 1.9.2+
* Rails ~> 3.1

## Setup Guide
Add the gem to your Gemfile.

  gem "bravo-generators", :group => :development
